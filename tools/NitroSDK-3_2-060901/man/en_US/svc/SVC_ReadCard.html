SVC_ReadCard

C Specification

#include <nitro/os/systemCall.h>
void SVC_ReadCard( s32 dmaNo, void* srcp, void* destp, s32 size );

Arguments

dmaNo DMA number (0 – 3) used for transmission
srcp Address on card
destp Console destination address
size Transmit size (Bytes)

Return Values

None.

Description

This function reads data from a card. If a DMA number other than 0 – 3 is passed, it will transmit using the CPU not DMA. However, in the case of the TEG board, the CPU always transmits. The card address must be aligned to a 512-byte boundary; the console address to a 4-byte boundary.
